Mini ChatGPT with Ollama
**** Overview
This project is a lightweight chatbot built with Ollama, designed to simulate a mini version of ChatGPT. It runs locally, making it easy to experiment with conversational AI without relying on cloud services.

****Features
Local AI inference with Ollama

Interactive chat interface (CLI or web-based)

Customizable prompts and personalities

Supports multiple Ollama models (e.g., llama2, mistral)# Mini-ChatGPT-Ollama
